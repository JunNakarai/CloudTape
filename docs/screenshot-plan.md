# CloudTape Screenshot Plan

Screenshots should make CloudTape feel calm, simple, and personal. Use real owned audio files with clean names. Avoid showing technical file paths, settings, diagnostics, or dense metadata.

## Visual Direction

- Spacious library screen
- Floating translucent mini player
- Clear shuffle playback state
- Light typography
- Comfortable touch targets
- Light and Dark Mode coverage
- No crowded file-management views

## Required States

### 1. Library Screen

Show a selected music folder with a concise list of tracks and the mini player visible at the bottom.

Caption idea: Your music, ready to play

### 2. Mini Player

Show active playback with artwork, title, pause button, and progress visible.

Caption idea: A quiet player that stays out of the way

### 3. Shuffle Playback

Show the expanded player with the shuffle control enabled and a track playing.

Caption idea: Start shuffle in one tap

### 4. Folder Selection

Show the first-launch empty state before folder selection, not a technical Files app screenshot.

Caption idea: Choose your iCloud music folder

### 5. Search

Show the search sheet with a short query and a few matching tracks.

Caption idea: Find a song quickly

### 6. Empty State

Show the unsupported or empty-folder state with the folder action visible.

Caption idea: Clear guidance when a folder is empty

### 7. Dark Mode

Show the library or expanded player in Dark Mode. Prefer active playback, not a blank state.

Caption idea: Calm in Light and Dark Mode

## Device Coverage

- iPhone 6.9-inch display
- iPhone 6.5-inch display if required by App Store Connect
- iPhone 5.5-inch display if required by App Store Connect
- Do not prepare iPad screenshots unless iPad support is intentionally enabled later

## Capture Checklist

- Use downloaded local/iCloud files so artwork and durations are stable.
- Avoid copyrighted album art in public screenshots unless rights are clear.
- Avoid personal names, private folder names, or sensitive filenames.
- Confirm text does not truncate awkwardly.
- Confirm the mini player does not cover key content.
- Confirm status bar, Dynamic Island area, and safe areas look clean.
- Capture both Light Mode and Dark Mode before final selection.
