# CloudTape App Review Notes Draft

CloudTape does not require an account.

## Fastest review path

1. Open CloudTape.
2. On the first screen, tap `サンプル音源を試す`.
3. This plays the bundled file `CloudTape-Demo-Audio.m4a`.
4. Use this path to verify playback, mini player behavior, background audio, and Lock Screen / Control Center controls without preparing an external library first.

## Normal user flow

1. Open CloudTape.
2. Tap `フォルダを選ぶ`.
3. Choose a folder from iCloud Drive or the Files app that contains supported audio files.
4. Tap any track or the shuffle button to begin playback.

## About the bundled sample

- The sample file is a short synthetic confirmation tone created for App Review.
- It does not contain third-party music, commercial music, or copyrighted music.

## In-app purchase note

- The optional in-app purchase `cloudtape.coffee.small` is a voluntary consumable support item.
- It does not unlock features.
- It does not change app behavior.
- Because it is consumable and grants no restorable entitlement, there is no Restore Purchases action.

## Privacy / business model note

- CloudTape has no account system.
- CloudTape has no ads.
- CloudTape has no analytics SDK.
- CloudTape does not track users.
