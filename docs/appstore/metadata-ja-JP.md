# CloudTape App Store Metadata (ja-JP)

## App名
CloudTape

## サブタイトル
iCloud Driveの音楽プレイヤー

## プロモーションテキスト
iCloud Driveにある自分の音楽を、静かでシンプルなプレイヤーで楽しめます。

## キーワード
音楽,プレイヤー,icloud,drive,オーディオ,ライブラリ,シャッフル,files,オフライン

## 説明
CloudTapeは、iCloud DriveやFilesアプリに保存した自分の音楽ファイルを再生するための、ミニマルな音楽プレイヤーです。

フォルダを選び、再生するだけ。CloudTapeは、すばやい再生開始、自然なシャッフル、落ち着いた画面を大切にしています。

アカウント、広告、トラッキング、SNS機能、レコメンド機能はありません。すでに音楽ファイルを持っている人のための、静かな個人用リスニングスペースです。

主な機能:
- iCloud DriveやFilesアプリ内の音声ファイルを再生
- フォルダ選択後すぐにシャッフル再生
- 選択したフォルダを一覧・検索
- バックグラウンド再生に対応
- ロック画面、コントロールセンター、ヘッドホン操作に対応
- 邪魔をしないミニプレイヤーと見やすい再生画面
- 手元の音楽をプライベートに再生

CloudTapeは意図的にシンプルです。ストリーミングサービス、SNS、音楽管理ツールではなく、自分で持っている音楽を静かに再生するためのアプリです。
