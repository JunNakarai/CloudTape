CloudTape

Support

CloudTape is a lightweight iOS player for owned music files stored in iCloud Drive or the Files app.

Common Checks

  • Confirm the selected folder is available in iCloud Drive or Files.
  • Wait for iCloud files to download locally before playback.
  • If folder access stops working, choose the folder again in the app.
  • Make sure the file format is supported by iOS playback.

Contact

For support, open an issue in the GitHub repository or contact the developer through the App Store support channel after release.
